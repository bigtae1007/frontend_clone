#### Mainpage
* 게시물 가져오기
#### funding
* 펀딩하기 - 선택한 reward와 수량을 체크하여 펀딩 금액 & 남은 수량 변경
#### 커뮤니티
* 기존 글 가져오기
* 새로 글 작성하기
* 글 수정하기
* 글 삭제하기

#### 댓글
